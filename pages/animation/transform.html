2D Transform

  1. translate
  2. rotate
  3. scale in X&Y axis for width and height
  4. skew in X&Y axis for width and height

hover effect can be used with all transform properties


translate

Translate allows an element to transform from its origin

translate property of transform accept rem and px in both ways. negative value with act on top and right translate

50px left, 100px bottom




Rotate

rotate property of transform allows an element to rotate from the origin

rotate accepts deg value, positive for clock and vice versa

rotates 20deg clockwise

Rotate in z-axis

like x and y axis an element can also be rotate in z-axis

rotates 90deg

Scale

Either scale-up or scale-down the size of the element

accepts times in one int, like 0-infinite, also to scale-down 0.9 to any numbers can be used

Scale X and Y can separately be used when needed

2x in both width, height and font is scaled





Skew

skews an element

Takes the value in deg like rotate, skew also accept separately X&Y value. when take 2 shorthand, first one is for x-axis second one for y-axis

20deg clockwise, 10 deg anti-clockwise



Matrix

Matrix is the shorthand for all the transform properties. takes 6 values. matrix allows all properties at once to transform an element.

shorthand of matrix: matrix(scaleX(),skewY(),skewX(),scaleY(),translateX(),translateY())
