HTML forms

HTML form requires a form tag as a container.

form tag attributes

  • action: what script to follow after the submission.
  • target: takes 5 values. target shows the response after the form is submitted.
    • _blank: the response displayed in a new tab.
    • _self: the response current tab.
    • _parent: the response displayed in the parent frame.
    • _top: the response displayed in the full body of the window.
    • framename: the response displayed in a named iframe.
  • method: takes two value
    • get: is not good way of sending sensitive data.
    • post
  • autocomplete: by default is of, however takes the value on, means completes the form based on previous data.
  • novalidate is a single attr with no value, when put means form validation will not happened

input & label

input & label works together most of the time. input is the input field and label is the title for that field. input changes depending on the type attr's value. the type values are following:

  • button: specifies as a button
  • text: specifies a small only-text input field
  • color: opens up a color selector option.
  • date: specifies a date selection option.
  • datetime-local: specifies date and time input field wit no local zone
  • email: specifies only email entry in the input field
  • file: specifies a file upload option.
  • hidden: this field is not visible to the user, often stores what database record that needs to be updated when the form is submitted.
  • image: input field for image upload.
  • month: input field for only month unlike date type.
  • number: input field for number only
  • password: input field for password. password field doesn't have visible character input.
  • radio: specifies radio selectable button
  • range: specifies range selection bar. the by default value is min=0 and max=100.
  • reset: a reset button for resetting all the value of other input field.
  • search: search field for the search options.
  • submit: acts as submit button for the form.
  • tel: specifies an input field for telephone number
  • time: input field for date and time.
  • url: input field for a web address.
  • week: input field for week no. and year.

input attributes

label

label is the title often goes before input and after radio buttons. for of the label should match the id of the input.

select

select is the dropdown list of options. select acts container for the option tags. select, option and optgroup works together to make functional input method for the user.

selected

by default the first option is selected. to define pre-selected option, selected attr is used

size: specifies the number of the visible options.

when the value is 3, it show 3 option without even any dropdown

multiple (empty-attr) allows the use to select multiple option at single time.

textarea

textarea is a multiple line input field. basically allow the user to input large data unlike names of passwords.

rows specifies the number of line in the text field is visible.

cols specifies the width of characters side wise.

button are clickable button.

fieldset is the related grouping of form elements

legend is the caption for the fieldset.

datalist:

similar to select/option, instead of using select/option combination, datalist goes inside of the input and type attribute is turned to sameValue as the datalist id.

output

output is the result of any calculation.
