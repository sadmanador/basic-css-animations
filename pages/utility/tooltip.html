Tooltip: hoverable quick info

following example is a basic Tooltip

Hover over me to see the tooltip. This is a simple Tooltip

positioning the tooltip

            .tooltip .tooltipText {
                top: -5px;      // to place the tooltip at the middle
                left: 105%;     // to add some the space to the right so that tooltip is detached from the text
            }
