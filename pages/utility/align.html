center align any element

This element is center aligned using margin: auto;
margin: auto; give equal left & right margin to horizontally align element.
text-align: center; to make the text of the element aligned at the center.

            div {
                margin: auto;
                text-align: center;
            }

center an image

            img {
                margin: auto;
                display: block;
            }

            position: absolute;
            right: 0px;
