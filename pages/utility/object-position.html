object-position:

If any img is object-fit: cover; the part of the img is clipped to fit the aspect ratio.
object-position: property make it easier to select to pick a specific part of the img.


horizontally pick the position of an img

object-position: hori% verti%;
The img viewport is selected left most part of the img using object-position: 10% 100%


vertically pick the position of an img

object-position: hori% verti%;
The img viewport is selected top most part of the img using object-position: 100% 10%
