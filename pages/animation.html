Briefing on animation

Animation allows the CSS to change the properties of an element for infinite time.
Some keyframes are needed to specify in order to use the CSS animation. keyframes holds what style an element will have at certain time.
animations unlike transitions doesn't require hover event or other events.

animation-direction: Animation in alternate directions

  1. normal
  2. reverse
  3. alternate: animation runs 1 whole cycle normal, next cycle reverse, and then vice versa
  4. alternate-reverse: opposite of alternate
