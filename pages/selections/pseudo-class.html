pseudo-class

pseudo-class are the special state of an element.

the style applied when the user:

  • mouse hovers over an element.
  • visits or unvisited links.
  • style the element when it's focused.

pseudo-class for a tags

  • a:link for default links.
  • a:visited for visited links.
  • a:hover when the link is hovered.
  • a:active to make a link look active.

pseudo-class for form input

:checked selects the checked input element.

:disabled selects any disabled input element

:enabled selects enabled input element.

:focus select input element with focus attr.

:in-range selects the input element with specific range

:out-of-range select the input element with value over of the range.

:valid select all the input element with a valid value.

:invalid selects all the input element with an invalid value.

:required selects the input element with "required attr

:optional selects element with no "required" attr

:read-only selects input element with read-only attr.
