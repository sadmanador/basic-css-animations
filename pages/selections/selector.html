General selectors

Css can select element by their own name. in such a case there is no extra thing necessary. 

            p {
                color: red;
            }

#id selector

in this case #tag followed by an ID name is needed.

            #idName {
                color: red;
            }

class selector

In case of class a period . followed by the class name is needed.

            .className {
                color: red;
            }

element along with class

element then period, then class to select a specific element with class name

This text is without any class, therefore unaffected.

This text will be affected, because it has a class.

            p.className {
                color: red;
            }

Universal html selector *

universal html element selector selects all the html elements. the following text will make all the element text red.

            * {
                color: red;
            }

HTML grouping selector.

grouping selector works when elements are grouped with comma sign.


This text is grouped

this text is grouped

this text isn't grouped

            h3, {
                color: red;
            }

Attribute selector

element with attr

following example is of an img element with the attr alt

        img[alt] {
            width: 100px;
        }

element, attr and specific attr value

Following exam shows an element div, with an attr width and value of 100px

        div[width="100px"] {

        }

attr with a specific word of value

following example shows any element with title attr and a specific word flower

        [title~='flower']{

        }

attr with value of a specific value or value and followed by a hyphen. the value has to be a whole word.

        [class|="top"]{

        }

In above case, top can have other words separated by hyphen.


specific attr which starts specific value. the value need not to a full word.

        [class^="top"] {

        }

above code can select anything that starts with top, weather top is a full word or not.


specific attr which ends specific value. the value need not to a full word.

        [class$="top"] {

        }

above code can select anything that ends with top, weather top is a full word or not.


specific attr which contain specific value. the value need not to a full word.

        [class*="top"] {

        }

above code can select anything that contain with top, weather top is a full word or not. the value could be in the middle of word.


form input type values.

following selection method selects the input field with specific type.

        input[type="text"] {

        }
        input[type="number"] {

        }

pseudo-class

pseudo-class are the special state of an element.

the style applied when the user:

  • mouse hovers over an element.
  • visits or unvisited links.
  • style the element when it's focused.

pseudo-class for a tags

  • a:link for default links.
  • a:visited for visited links.
  • a:hover when the link is hovered.
  • a:active to make a link look active.

child selection.

:first-child select the first child of a parent element.

        ul li:first-child {

        }

:first-of-child elements that is the first element of it's parent

should select every p tag in all the div

        div p:first-of-type {

        }

:last-child

selects the last child of the parent

:last-of-type

selects every last child of similar parent element.

:nth-child(n)

selects the nth number of child from its parent.

  • apple
  • apple
  • apple
  • apple
  • apple

in above case, 3th child is selected.


:nth-last-child(n)

selects the nth number of child from last of its parent.


:only-child

select the parent that have only one child.


pseudo-class for form input

:checked selects the checked input element.

:disabled selects any disabled input element

:enabled selects enabled input element.

:focus select input element with focus attr.

:in-range selects the input element with specific range

:out-of-range select the input element with value over of the range.

:valid select all the input element with a valid value.

:invalid selects all the input element with an invalid value.

:required selects the input element with "required attr

:optional selects element with no "required" attr

:read-only selects input element with read-only attr.


pseudo-elements

pseudo-element are use to style specific parts of an element.

  • style the first letter, or line of an element.
  • insert content before or after of the content of an element.

::first-line add specific style to the first line of an element.

resizing of viewport may not affect the first-line
