transition

transition attr and property need to be passed on in non-hoverable state and :hover will only contain the specific changes

div { transition: width 10s }

div:hover { width: 100px }

The width will transit from origin to 1000px in 10sec

transition 2 values at once

two values separated by commas can be executed

width to 1000px and height to 400px upon hover
