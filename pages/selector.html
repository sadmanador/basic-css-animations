Css selector

  • general selector.
  • Attribute selector.
  • pseudo-class.
  • pseudo-element.

General selectors

Css can select element by their own name. in such a case there is no extra thing necessary. 

            p {
                color: red;
            }

#id selector

in this case #tag followed by an ID name is needed.

            #idName {
                color: red;
            }

class selector

In case of class a period . followed by the class name is needed.

            .className {
                color: red;
            }

element along with class

element then period, then class to select a specific element with class name

This text is without any class, therefore unaffected.

This text will be affected, because it has a class.

            p.className {
                color: red;
            }

Universal html selector *

universal html element selector selects all the html elements. the following text will make all the element text red.

            * {
                color: red;
            }

HTML grouping selector.

grouping selector works when elements are grouped with comma sign.


This text is grouped

this text is grouped

this text isn't grouped

            h3, {
                color: red;
            }

Attribute selector

element with attr

following example is of an img element with the attr alt

        img[alt] {
            width: 100px;
        }

element, attr and specific attr value

Following exam shows an element div, with an attr width and value of 100px

        div[width="100px"] {

        }

attr with a specific word of value

following example shows any element with title attr and a specific word flower

        [title~='flower']{

        }

attr with value of a specific value or value and followed by a hyphen. the value has to be a whole word.

        [class|="top"]{

        }

In above case, top can have other words separated by hyphen.


specific attr which starts specific value. the value need not to a full word.

        [class^="top"] {

        }

above code can select anything that starts with top, weather top is a full word or not.


specific attr which ends specific value. the value need not to a full word.

        [class$="top"] {

        }

above code can select anything that ends with top, weather top is a full word or not.


specific attr which contain specific value. the value need not to a full word.

        [class*="top"] {

        }

above code can select anything that contain with top, weather top is a full word or not. the value could be in the middle of word.
