grid

To added grid layout, First grid container and grid-item are added.

grid container and grid-items are added by adding simple class to div elements.


configuration of container.

  • display: grid; = grid layout as block.
  • display: inline-grid; = grid layout as inline component.
  • gap: value in px; = sets gap between grid items.
  • colum-gap: px; = sets gap value between items for colum.
  • row-gap: px; = sets gap value between items for row.
  • grid-colum-start: 1; = tells that grid item will start from line 1.
  • grid-colum-end: 4; = tells that grid item will end at line 4 and make 3 grid items.
  • grid-row-start: 1; = tells that grid item will start from line 1.
  • grid-row-end: 4; = tells that grid item will end at line 4 and make 3 grid item long row wise.
  • grid-template-column: auto auto px px = create 4 column, two automatically resize, other 2 are fixed with specific px
  • grid-template-row:auto auto px px = create 4 row, two automatically resize, other 2 are fixed with specific px

justify-content for grid container.

  • justify-content: space-evenly; = give space to item in around them horizontally.
  • justify-content: space-around; = give space to item in around them horizontally.
  • justify-content: space-between; = give space to item in between them horizontally.
  • justify-content: center; = will align the items horizontally at the center.
  • justify-content: start; = will align the items horizontally at the beginning.
  • justify-content: end; = will align the items horizontally at the end.

align-content: helps to align items of grid vertically.

  • align-content: space-evenly; = give space to item in between and around them vertically.
  • align-content: space-around; = give space to item in around them vertically.
  • align-content: space-between; = give space to item in between them vertically.
  • align-content: center; = align the items vertically in the middle.
  • align-content: end; = align the items vertically in the bottom.
  • align-content: start; = align the items vertically in the top.

grid item

these settings are applicable to grid items, to configure them.

  • grid-column: 1 / 5; = first value refers the start line of start point, second refer to the end of the item and which line.
  • grid-column: 1 / span 3; = first value refers the start line of start point, second refer span 3 column.
  • grid-row: 1 / 5; = first value refers the start line of start point, second refer last line where grid item will end at row 5.
  • grid-row: 1 / span 3; = first value refers the start line of start point, second refer span 3 row.
  • grid-area: 1 / 3 / 1 / 4; = shorthand property for grid-row-start, grid-column-start, grid-row-end and the grid-column-end.
  • grid-area: 1 / 1 / span 3 / span 4; = used with span values.

Naming the grid item

following item takes the name mainArea, spans 4 grid column.

            .item2 {
                grid-area: mainArea:
            }

            .container {
                grid-template-areas:  'mainArea mainArea mainArea mainAre';
            }

adding name with those who have no name

a period sign (.) represents the grid item with no name. 3 periods mean that the rest of the column is occupied by item those have no name.

            .item2 {
                grid-area: mainArea;
            }

            .container {
                grid-template-areas: mainArea mainArea . . .;
            }

logical way of using grid name

            .item1 { grid-area: header; }
            .item2 { grid-area: menu; }
            .item3 { grid-area: main; }
            .item4 { grid-area: right; }
            .item5 { grid-area: footer; }

            .grid-container {
            grid-template-areas:
            'header header header header header header'
            'menu main main main right right'
            'menu footer footer footer footer footer';
}
